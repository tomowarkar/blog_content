# A
### 問題
```
整数Nが与えられます。高橋君は、N以下の正整数の中から等確率で1つを選んでaとします。このとき、aが奇数である確率を答えてください。
```
### 制約
```
1 ≤ N ≤ 100
```
### 解答
```go
package main
 
import "fmt"
 
func main() {
	var a int
	fmt.Scan(&a)
	fmt.Println(1.0-float64(a/2)/float64(a))
}
```
